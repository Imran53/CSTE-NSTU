# CSTE-NSTU
Student Portal Android app for the Students of CSTE, NSTU.

**Key features of CSTE, NSTU app includes:** 

* Profile
* Teachers 
* Students 
* Syllabus 
* Class Schedule 
* Books 
* Questions 
* Notices & Events 
* Forum 
* Alumni 
* Necessary Groups 
* Useful Videos 
* Interview Guideline 
* CP resources 
* Notification
